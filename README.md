# Memories App

A Simple Memories App built on MERN Stack.

## Configuration

Create .env file in root folder. Add the following configurations:

- MongoURI - MongoDB Database URL

## Installation

```bash
$ npm install
$ npm run client-install
```

## Running the app

```bash
# Run server & client in development
$ npm run dev

# Run server only
$ npm start

# Run client only
$ npm run client
```
